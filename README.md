# Task Manager

A full-stack **Task Manager** web application that allows users to manage their daily tasks efficiently. Users can create, update, delete, and track the status of their tasks through a simple and responsive interface.

This project demonstrates complete **CRUD operations**, REST API integration, and full-stack development using modern web technologies.
